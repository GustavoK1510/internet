Resposta: As regras CSS são declarações de estilização do texto, que podem ser usadas de diversas formas. Elas possuem algumas partes específicas: Seletor: parte que você define quais elementos serão afetados por aquela regra. Exemplos: elementos são selecionados somente pelo nome, "#" são usadas para selecionar pelo id e "." para selecionar pela classe. Bloco de declarações: Definido com "{}" após o seletor. Onde são escritas as estilizações que serão usadas. Propriedade: São as regras em si, sempre seguidas por ":". Podem definir padrões de cor, posicionamento, entre várias outras características. Valores: São os valores que serão usados nas padronizações. Não necessariamente é um valor numérico, podem ser cores, estilos de fonte, etc. Exemplos: pseudoelemento{ color: red; font-size: 12pt; font-family: Arial, Helvetica, sans-serif; } .pseudoclasse{ padding-left: 20px; border: 1px solid black; width: 50px; height: 300px; }
